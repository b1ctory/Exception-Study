#### Custom View를 만들려면 어떻게 해야하나?

1. Xib 이용
   1. 눈으로 보면서 쉽게 레이아웃 작업 가능
   2. 뷰 컨트롤러에 추가적인 코드작업 필요 없음
   3. 레이아웃의 에러를 찾기 쉬움
   4. 재사용성 Good
   5. 충돌시 해결해주어야 할 문제가 많음
   6. 코드로 작성한 UI에 비해서 오버헤드가 크다 (필요시 메모리 로드 > 파싱)
2. 코드로만 작성
   1. 어떤 일이 일어나는지 알 수 있음
   2. 개발자 마음대로 제어 가능 (동적 레이아웃)
   3. 충돌 시 해결 쉬움
   4. 재사용성
   5. 개발 기간이 오래걸림
   6. 에러 찾기가 어려움
   7. 코드 리팩토링 어려움
